practice core Go concepts such as variables, constants, loops, functions, conditionals, input validation, slices, and basic string operations.

Features
Welcomes users and shows total and remaining conference tickets

Takes user input: first name, last name, email, and number of tickets

Validates:

Name length (at least 2 characters)

Email format (must contain @)

Ticket count (positive and not more than remaining tickets)

Books tickets and stores bookings in a slice

Extracts and prints the first names of all attendees

Stops booking when all tickets are sold out
